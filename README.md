## 🎨 Project Overview
This project was developed for the **Art_Big_Data** course at **Sungkyunkwan University (SKKU)**. It serves as a bridge between artistic intuition and quantitative analysis.

As a student navigating the intersection of art and data, I built this dashboard to transform the often subjective world of fashion into a visual "Data Hub." By comparing historical silhouettes (specifically from the **Qing Dynasty**) with modern **Haute Couture**, this tool quantifies the "Zeitgeist" and provides actionable insights for designers and researchers.

---

## 🧭 Project Rationale (The "Why")
- **Bridging Art and Logic:** Proving that "Data-Informed Creativity" is the future of fashion.
- **Market Intelligence:** In a saturated market, understanding consumer sentiment and sustainability metrics is a mechanical necessity for brand survival.
- **Strategic Visualization:** Translating complex big data into a "Gallery-like" interface for creative professionals.

---

## ✨ Key Features
- **Aesthetic Radar:** Side-by-side trait comparison (Sleeve volume, embroidery density, etc.) between historical eras and modern luxury designers.
- **Sentiment & "Vibe" Tracker:** Interactive gauges measuring brand heat and marketing psychology (focusing on authenticity and prestige).
- **Sustainability Scorecard:** Metric-based visualizations of the environmental impact of luxury textiles.
- **Trend Diffusion Map:** A geographic heatmap showing the origin and spread of K-Style and global fashion keywords.
- **Interactive Gallery:** Real-time filtering of collection images paired with metadata analytics.

---

## 🛠️ Tech Stack & Methodology
- **Framework:** [Streamlit](https://streamlit.io/) (Web Dashboard)
- **Language:** Python 3.x
- **Analysis:** Pandas, NumPy, Scikit-learn (K-means Clustering)
- **Visualization:** Plotly, Altair
- **Data Sources:** Vogue Runway, Met Museum Open Access API, Kaggle Fashion Datasets
